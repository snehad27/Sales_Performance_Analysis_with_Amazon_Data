# Sales_Performance_Analysis_with_Amazon_Data
The analysis utilizes a comprehensive dataset of Amazon sales transactions spanning a specific period (January 2020 to December 2024) across various regions and product categories. Key data attributes include Order ID, Order Date, Product Name, Category, Quantity Sold, Unit Price, Total Sales Amount, Payment Method, and Order Status.

## Tools used
Excel for data cleaning
Tableau for data Visualization.
